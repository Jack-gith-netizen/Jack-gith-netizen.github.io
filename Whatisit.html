Facts about CSS Transforms, Transitions, and Animations

1. CSS Transforms

CSS transform allows you to modify an element's position, size, rotation, or shape in 2D or 3D space.

  • Move: Moves an element from its current position.
  • Scale: Changes the size of an element.
  • Rotate: Rotates an element.
  • Skew: Slants the element.
  • 3D Transforms: Creates depth effects using properties like perspective.

Example

When hovering over the box:

  • It moves 200px to the right (translateX(200px)).
  • It rotates 45 degrees (rotate(45deg)).
  • It scales by 1.5 times (scale(1.5)).

2. CSS Transitions

CSS transitions allow smooth property changes over time.

  • transition-property: Specifies which CSS property will change.
  • transition-duration: Sets how long the transition will take.
  • transition-timing-function: Defines the speed curve of the transition.
  • transition-delay: Adds a delay before the transition starts.

Example

When hovering over the box:

  • The background color changes to coral over 1 second.
  • The element rotates by 180 degrees over 3 seconds.

3. CSS Animations

CSS animations offer more flexibility than transitions.

  • @keyframes: Defines the keyframes of an animation.
  • animation-name: Specifies the animation's name.
  • animation-duration: Defines how long the animation lasts.
  • animation-timing-function: Controls the animation speed curve.
  • animation-iteration-count: Defines how many times the animation repeats.

Example

Animation breakdown:

  • At 0%: The box is in its original position with a teal color.
  • At 50%: It moves right, rotates 180 degrees, and changes to coral.
  • At 100%: It returns to its original position and color.

Summary of Differences

  • Transforms: Apply 2D or 3D transformations to an element (scaling, rotating, translating).
  • Transitions: Smoothly animate property changes when an element's state changes.
  • Animations: Allow complex effects with keyframes and precise timing control.
